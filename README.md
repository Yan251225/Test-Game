# 🌸 星辰学院 — 乙女游戏

一款现代校园风格的网页乙女游戏，纯 HTML/CSS/JavaScript 实现。

## 🎮 如何游玩

1. 双击 `index.html` 在浏览器中打开即可游玩
2. 或者用任意 HTTP 服务器托管本目录（推荐，以避免跨域问题）：
   ```bash
   # Python
   python -m http.server 8080

   # Node.js
   npx serve .
   ```
   然后访问 `http://localhost:8080`

## 📖 故事简介

你是一名刚转入「星辰学院」的高二学生**苏念念**。在新学校的第一天，你将遇到三位性格迥异的男生：

| 角色 | 身份 | 性格 |
|------|------|------|
| 🔵 **陆辰逸** | 学生会长 | 温柔可靠，外表完美但内心孤独 |
| 🟣 **顾晏时** | 图书馆常客 | 安静神秘，有不为人知的秘密 |
| 🟠 **林笑寒** | 体育部王牌 | 阳光开朗，热情直率 |

## 💕 游戏系统

- **好感度系统**：你的选择会影响与每位角色的好感度（0-100）
- **多分支剧情**：序章 → 共同线 → 选择角色路线
- **多结局**：
  - 好感度 ≥ 80：💕 好结局
  - 好感度 40-79：🌸 普通结局
  - 好感度 < 40：💔 坏结局
- **存档/读档**：6 个存档槽位 + 快速存档
- **角色图鉴**：查看角色详细资料

## 🎯 操作说明

- 点击对话框或按 `Enter`/`Space` 继续对话
- 文字显示中途点击可跳过打字效果
- 点击选项做出选择
